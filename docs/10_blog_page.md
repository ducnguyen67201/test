# Blog Page

## **Purpose**
A space for sharing insights, tutorials, and updates from the CyberOctopus team — designed with visual consistency and easy readability.

---

## **Layout Overview**

Each blog entry is displayed as a **card**, arranged in a responsive grid (3 per row on desktop, 1–2 per row on mobile).

---

### **Card Components**

| Element | Description |
|:---------|:-------------|
| **Thumbnail Image** | Lab-themed visuals, AI-generated artwork, or relevant screenshots |
| **Category Tag** | Small uppercase label, e.g. “AI / Research”, “Exploit / How-To”, “News / Labs” |
| **Title** | Bold white text, 1–2 lines max |
| **Excerpt** | Light gray text, short preview (1–2 sentences) |
| **Hover Effect** | Card subtly lifts with a **soft glow** to emphasize interactivity |

---

### **Design Notes**

- Background uses a dark gradient to make white text stand out.  
- Cards have rounded corners and subtle shadow.  
- Hover glow color reflects the CyberOctopus brand palette (tea
